Flexbox Cheat Sheet

  • display: flex;: Creates a flex container.
  • justify-content: Aligns items horizontally (e.g., center, space-between, space-around).
  • align-items: Aligns items vertically (e.g., center, flex-start, flex-end).
  • flex-direction: Sets the direction of items (e.g., row, column).
  • flex-wrap: Allows items to wrap onto multiple lines.
  • align-self: Allows individual items to override the container's alignment.
  • order: Changes the order of items in the flex container.
  • flex-grow: Specifies how much a flex item should grow relative to the others.
  • This cheat sheet provides practical examples of the most commonly used Flexbox properties, organized in order of importance.
